# 🌐 Enterprise Network Lab (3-Floor Architecture)

## 🚀 Project Overview

This project simulates a **real-world enterprise network** designed using **CCNA fundamentals** and a **CCNP-level approach**.

It represents a **3-floor office network architecture** with proper segmentation, routing, and wireless connectivity.

---

## 🏢 Network Design

* Hierarchical Design (Core, Distribution, Access)
* Multi-floor enterprise structure
* Scalable and modular architecture

---

## 🌐 Features Implemented

* Inter-VLAN Routing
* VLAN Trunking (802.1Q)
* Static Routing
* OSPF (Dynamic Routing)
* DHCP Configuration
* Wireless Access Points (WLAN)
* WAN Simulation (Router-to-Router)

---

## 🔧 Technologies Used

* Cisco Packet Tracer
* Routing Protocol: OSPF
* Switching: VLAN, Trunking
* Services: DHCP, ARP, ICMP
